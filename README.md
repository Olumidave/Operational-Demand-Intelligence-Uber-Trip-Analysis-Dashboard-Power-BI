# Uber Trip Analysis Dashboard (Power BI)

This project analyzes Uber trip data using Power BI to uncover booking trends, revenue performance, and trip efficiency.

## Key Features
- Dynamic KPI tracking (Total Bookings, Total Revenue, Avg Trip Distance)
- Time-based metrics
- Drill-through analysis for detailed trip insights
